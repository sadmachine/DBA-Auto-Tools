Overview

Before performing any troubleshooting, its important to verify the integrity of the program itself and any of its resources.

Resources that the program relies on to function properly include:

  • Filesystem
  • Shortcuts
  • Config Files

In addition, its important to check for certain signatures that the program is operating properly, such as

  • the Tray Icon/Menu associated with the program
  • the Processes that should be running
