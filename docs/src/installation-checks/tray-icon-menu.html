Summary

If the Tray Icon does not appear in the task bar (bottom-right side) that means the program is not running. Locate the desktop icon and reopen the program.

Introduction

The Tray Icon is one of the ways to verify that DBA AutoTools is running in the background. It should appear in the bottom right corner of the screen on the taskbar, but may be hidden. The two possible locations it should appear can be found below:

  • On the taskbar:
  • Hidden in the icon menu:

Looking for the icon is usually the fastest way to tell if the program is running. If the icon is missing, that means that the program is not running properly, or at all. If this is the case, you should be able to find a shortcut to the program on the desktop; double-click this shortcut to restart the program, and verify that the tray icon appears in the taskbar.

Tray Menu

The Tray Menu is what comes up when you right click the Tray Icon. If the menu does not appear when you right-click the tray icon, this might indicate there is a problem with the program.
