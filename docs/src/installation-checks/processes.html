Summary

Verify that the following processes are running. If they aren't the program needs to be restarted.

  • DBA AutoTools.exe
  • QueueManager.exe

Introduction

There are certain processes that should exist within Windows Task Manager if the program is successfully running on a machine. If any of the processes is missing, the program may not run as expected, or at all.

Expected Processes

The two processes that should be running are:

  • DBA AutoTools.exe
  • QueueManager.exe
