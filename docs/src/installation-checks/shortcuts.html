Summary

Verify that shortcuts to the main executable (DBA AutoTools.exe, found in the installation directory) exist in the following locations:

  • shell:common startup (usually translates to C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup)
  • shell:common desktop (usually translates to C:\Users\Public\Desktop)

Introduction

There are several places that shorcuts to the main program can be placed for convenience. The most important location is within the Startup Folder (alias shell:common startup) for the computer. Placing the shortcut here ensures that anytime the computer starts up, the program will launch and run in the background, waiting for DBA Manufacturing to be opened so it can be displayed.

Another common location to put the shortcut is within the Public Desktop (alias shell:common desktop) so that is appears on the desktop of all users that sign onto the machine.
